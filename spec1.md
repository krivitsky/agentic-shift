# Agentic Shift Conference Website - Spec 1

## Overview
Conference website styled as a Claude Code CLI terminal.

## Stack
- JavaScript, Node

## Look & Feel
- Dark mode terminal (Claude Code CLI aesthetic)
- Monospace font, dark background, light text
- Full-screen terminal on entry

## People
- Co-producers: Alexey (gray beard man) and Martin (red-haired man)

## Location
- Munich, Germany
- June 27, 2026

## Entry Animation Sequence
1. Dark terminal appears fullscreen
2. Types out (CLI typing effect): `welcome to agentic shift conference`
3. Draws ASCII contour of Munich's Frauenkirche (the two iconic towers)
4. Types: `Munich, June 27, 2026`
5. Shows a prompt (e.g. `>`)
6. Auto-types `/help`
7. Prints available commands:
   - `/program`
   - `/registration`
   - `/contact`
8. Waits for user input at the prompt
